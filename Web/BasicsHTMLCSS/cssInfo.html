What is CSS ?

CSS is abbreviated from Cascading Style Sheet, and is used for styling the webpage.

We have three types of CSS :-

Inline > Internal > External
  • Inline CSS :- The styling that we do instantly inside the tag line of the HTML page by using the style attribute is called Inline CSS.
  • Internal CSS :- The styling that we do in the HTML page but by using the style tag in the head of HTML is called Internal CSS.
  • External CSS :- The styling that we do externally by creating a different file(like style.css) and link this page to the HTML page via the link attribute is called External CSS.

Since HTML code is ran from top to bottom so as the link tag that links our external css is written first inside our head tag, then comes internal styling css and then inline css so internal overrides external CSS and inline overrides everyone.

  12. nth-of-type selector selects a specific element based on its type and order in another element - or even or odd instances of that element.
    (Eg:(i) div:nth-of-type(2) selects the second instance of a div.
    (ii) .example:nth-of-type(odd) selects all odd instances of a the example class.)
    (iii) plate:nth-of-type(3n) selects all the elements in basis of multiple of 3, i.e 3rd line then 6th line then 9th and so on
  13. nth-of-type selector with formula -- The nth-of-type formula selects every nth element, starting the count at a specific instance of that element.(Eg:- span:nth-of-type(6n+2) selects every 6th instance of a span, starting from (and including) the second instance.)
  14. only-of-type selector -- select elements that are the only ones of their type within of their parent element(Eg:- p span:only-of-type selects a span within any p if it is the only span in there.)
  15. last-of-type selector selects each last element of that type within another element.
    (Eg :-(i) div:last-of-type selects the last div in every element.
    (ii) p span:last-of-type selects the last span in every p.
  16. Negation Pseudo-class -->> :not(anything) You can use this to select all elements that do not match selector "X".
    Examples:-(i) :not(#fancy) selects all elements that do not have id="fancy".
    (ii) div:not(:first-child) selects every div that is not a first child.
    (iii) :not(.big, .medium) selects all elements that do not have class="big" or class="medium".
