Things learnt in HTML

  1. h1-h6 for heading
  2. p for paragraph
  3. img for images, where we can pass the source in two ways:-
    • Full Path (Eg. /users/bishal/desktop/html/bird.jpg)
    • Relative Path (Eg. /html/bird.jpg)
  4. div tags -> for aligning many items in one container.
  5. strong or bold (strong was in old HTML, now it's bold)
  6. em or i -> for italics(em was in old, i is in new)
  7. a href -> as anchor tag(target equalsto underscore blank)
  8. table tag for creating table, you can also use border attribute in table for the bordering/outlining table
  9. thead and tbody of table ->(tr and th in thead , tr and td in tbody) tr as table row and th as table head and td as table data
  10. Shortcut :- if you want 4 td tags or any tags in multiple form than you can directly multiply that specific tag * no of times you want it(Eg:- td*4, this gives you 4td's at once)
  11. colspan and rowspan are also attributes that can be used in th/td if we want two or more rows to combine under one subtopic. Eg: I've shown in the 2nd table below
  12. caption tag is used to give caption/topic/heading on what is the table about.(It is recently introduced in HTML5).
  13. Block elements are those which takes full block or those tags that execute in new line are said to be as block elements
  14. Inline elements are those tags which execute side by side or those which does not cover the whole line and only covers till whatever content it has
  15. (Eg tags of block element:- div, p, table, h1 ... h6 etc are some examples of the block element)
  16. (Eg tags of inline element:- th, td, img, span etc are some examples of the inline element)
  17. span tag is also an inline element
